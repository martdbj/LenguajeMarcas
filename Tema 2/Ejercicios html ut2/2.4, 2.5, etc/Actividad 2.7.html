Breve introducción a HTML y la WWW

¿Qué es HTML?

Para empezar a plantearnos esta pregunta primeros nos tenemos que preguntar por qué surgió HTML, esto se remonta al 1980, cuando el científico Tim Berners-Lee propuso un nuevo sistema de hipertexto para poder compartir documentos. A partir de esto HTML a ido evolucionando hasta encontrarse en la versión actual, HTML5.


Creación de la WWW

Tim Berners-Lee habiendo hecho HTML se propuso la Creación del proyecto de la WWW, empezó desarrollando en el año 1989, tras unos años de desarrollo salió al público, con ello el lanzamiento de navegadoras empezó a surgir como con por ejemplo ViolaWWW o Mosaic.

Para explicar un poco como funciona la WWW, podemos empezar explicando que el primero paso consiste en traducir parte del servidor URL en una dirección IP, usando una base de datos distribuida de Internet conocida como DNS.Esta dirección IP se considera necesaria para poder contactar con el servidor web y poder enviar paquetes de datos.

El siguiente paso es enviar una petición HHTPS al servidor web solicitando los recursos, normalente se solicita primero el texto HTML que es inmediatamente analizado por el navegador, que después pide todo lo demás restante de la página.

AL recibir estos ficheros que hemos solicitado desde el servidor web, el navegador renderiza la página tal y como se describe en el código HTML o los otros lenguajes que la página pueda incorporar. Por último se incorporan las imágenes y otros recursos para terminar de cargar la página web que el usuario ve en su pantalla.
